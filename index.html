Space Mammal 128

Objective

Survive as long as possible while destroying as many asteroids as you can.

Features

  • The player controls a ship rendered and textured from an .obj file.
  • Simple Axis-Aligned Bounding Box (AABB) collision detection

Controls

  • Move ship: arrow keys
  • Shoot: space bar

How to run

Run usual java -cp command but on the Main.class file in the src/game/ folder
